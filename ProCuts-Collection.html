Helpful Tools

When developing shortcuts, itâ€™s crucial to use your resources to help improve workability. We use many tools to help prefect our shortcuts. The following are some very useful tools we use that we think are important to share. We did not develop nor are we affiliated with any of these helpful tools.

Link Picture

We used Link Picture quite a bit in our shortcuts. Link picture is a website that allows you to upload a picture and obtain the direct link to it. We use these links to save space in our shortcuts (instead of large base64 encodes). We usually used them in the HTMLs.

Phone Mock-Ups

To make your image examples look professional, you can use this website. It puts your screenshot examples into a phone mock-up. As you see, we used these to show what our shortcutâ€™s end results can be. The process is extremely easy and there are an abundance of devices to choose from.

Dictionary Action Builder

Turn your raw JSON files into a shortcut "Dictionary" action with this shortcut made by @glubyte. This can be extremely helpful when working with large dictionaries or other coding languages. Itâ€™s definitely a valuable addition to your development tools collection. This shortcut was used in the Font Switcher shortcut to create a dictionary faster than putting in each individual letter.

Copy And Paste A Shortcut

Easily copy actions from one of your shortcuts into another using this tool by @robric18. This is helpful in many cases, such as if you use a snippet of code in most of your shortcuts (like an updater). Instead of copying it action by action, just run this shortcut to copy and paste all the actions at once. Although a bit excessive in the settings, it is very easy to use. We would advise using the quick copy action for all of your needs.

Coloured vCards

Create coloured or gradiented vCard menus in shortcuts without large base64 encodings

Coloured vCards provides a simple way to not only create colourful vCard menus, but use less device storage by avoiding large base64 encodings. The snippet uses JavaScript to craft the menu which is faster than the average vCard and the very customizable input dictionary will provide countless options that surpasses the average vCard menu.

Coloured vCards supplies the snippet to make these vCard menus and a documentation to learn the input dictionary. Coloured vCards can be used for many purposes and ultimately makes the menu more appealing and distinguishable.

The icons in the menu can be mono-coloured or dou-coloured, gradiented, linear or radical, and even rotated. You can also choose the titles and subtitles of the items and even the email for unseen information.
